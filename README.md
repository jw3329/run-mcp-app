# 🏃 Garmin Running Club — download

Your own Garmin running data as a **dashboard** + **AI-assistant tools** — running
entirely on your own computer. No account here, no sign-up; you log in with your own
Garmin and your data never leaves your machine.

## Download & run
1. Grab the file for your computer from **[Releases](../../releases/latest)** and unzip:
   - **Mac** → `run-mcp-macOS.zip`
   - **Windows** → `run-mcp-Windows.zip`
   - **Linux** → `run-mcp-Linux.zip`
2. **Open `GarminRunningClub`** (first time: Mac → right-click → Open; Windows → "More info" → Run anyway).
3. Your browser opens a **login page** → enter your Garmin email/password (+ 2-step code if asked) → **Connect**.

Your dashboard loads. Re-open the app anytime — it remembers your login.

## Use it in an AI assistant (optional)
On the dashboard, click **Connect to Claude Desktop** or **Connect to Gemini CLI** (one click,
then restart that app). **ChatGPT** is supported too via the **Connect to ChatGPT** button
(it opens a temporary public tunnel — keep the app open and Stop it when done).
